The roboShieldMega Arduino Mega Robot shield

An Arduino Mega roboShield, back from 2010.

Features:

The shield breaks out the Arduino Mega pins to the following modules:

  • Six DC motors (controlled by XX)
  • Eight analog sensors
  • Four bumpers (push-buttons)
  • Four ultrasonic sensors (model XX)
  • Bluetooth module (for module XX)
  • Colour sensor (model XX)
  • IR beacon (Model XX)

In addition, it features a separate motor power supply that can be tied to the logic supply via a jumper and a reset button.
